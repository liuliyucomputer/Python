# Python入门课程

## 一、Python简介

### 1. Python的历史和特点
Python是一门1991年由Guido van Rossum发明的编程语言。它语法简单，易学易用，代码可读性强。Python跨平台，能在Windows、Linux、Mac等系统上运行。它有丰富的第三方库，能快速开发各种应用。

### 2. Python的应用领域
Python可以做网站开发、数据分析、人工智能、自动化脚本、爬虫、科学计算、桌面应用等。只要你能想到的编程领域，几乎都能用Python实现。

### 3. Python的安装和环境配置
- Windows：去官网python.org下载，双击安装。
- Linux/macOS：一般自带，也可用命令`sudo apt install python3`。
- 安装好后，命令行输入`python`或`python3`，出现版本号说明安装成功。
- 推荐用`pip`安装第三方库，如`pip install numpy`。
- 建议用虚拟环境隔离项目：`python -m venv venv`，激活后再安装依赖。

## 二、基本语法和数据类型

### 1. 变量和赋值
变量就是用来存数据的名字。Python变量不用声明类型，直接赋值即可。
```python
name = "小明"
age = 18
```
注意：变量名只能用字母、数字、下划线，不能以数字开头，区分大小写。

### 2. 数字类型：整数、浮点数、复数
- 整数：没有小数点，如`a = 10`
- 浮点数：有小数点，如`b = 3.14`
- 复数：形如`c = 2 + 3j`
```python
x = 100
pi = 3.1415
z = 1 + 2j
print(type(x), type(pi), type(z))
```
注意：复数的虚部用j结尾。

### 3. 字符串操作
字符串就是一串字符，用单引号或双引号包起来。
```python
s = "Hello, Python!"
print(s[0])      # H
print(s[-1])     # !
print(s[2:5])    # llo
print(s + " 学习")  # 拼接
print(s * 2)     # 重复
```
注意：字符串不可变，不能直接修改某个字符。

### 4. 列表和元组
- 列表：有序、可变，用[]。
- 元组：有序、不可变，用()。
```python
lst = [1, 2, 3]
tpl = (4, 5, 6)
lst.append(4)
print(lst)
# tpl[0] = 10  # 错误，元组不能改
```
注意：元组只有一个元素时要加逗号，如`(1,)`。

### 5. 字典和集合
- 字典：键值对，用{}。
- 集合：无序、无重复元素，用{}或set()。
```python
d = {"name": "小明", "age": 18}
print(d["name"])
d["city"] = "北京"
print(d)
s = {1, 2, 3}
s.add(4)
s.remove(2)
print(s)
```
注意：字典的键必须不可变类型，集合不能有重复元素。

### 6. 数据类型转换
```python
num = int("123")
flt = float("3.14")
str1 = str(456)
lst = list((1, 2, 3))
tpl = tuple([4, 5, 6])
set1 = set([1, 2, 2, 3])  # {1, 2, 3}
```
注意：字符串转数字必须内容能转换，否则报错。

## 三、控制流

### 1. 条件语句（if, elif, else）
```python
score = 85
if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```
注意：冒号不能少，缩进必须统一（推荐4个空格）。

### 2. 循环（for, while）
```python
for i in range(1, 6):
    print(i)

n = 1
while n <= 5:
    print(n)
    n += 1
```
注意：for循环常配合range，while循环要有结束条件，否则死循环。

### 3. break和continue语句
```python
for i in range(1, 10):
    if i == 5:
        break  # 提前结束循环
    print(i)

for i in range(1, 6):
    if i % 2 == 0:
        continue  # 跳过偶数
    print(i)
```

### 4. pass语句
pass什么都不做，常用于占位。
```python
def func():
    pass  # 以后再写
```

## 四、函数

### 1. 函数定义和调用
函数就是一段可以重复使用的代码。
```python
def greet(name):
    print(f"你好，{name}！")

greet("小明")
```
注意：函数名后面要加括号，参数可有可无。

### 2. 参数和返回值
```python
def add(a, b):
    return a + b

result = add(3, 5)
print(result)
```
注意：用return返回结果，函数没有return默认返回None。

### 3. 变量作用域
- 局部变量：只在函数内部有效。
- 全局变量：在整个文件都能用。
```python
x = 10

def foo():
    y = 5
    print(x, y)

foo()
print(x)
# print(y)  # 错误，y只在函数内有效
```
注意：在函数内修改全局变量要用global关键字。

### 4. Lambda函数
Lambda是匿名函数，适合写简单的一行函数。
```python
f = lambda x, y: x + y
print(f(2, 3))  # 5
```
注意：lambda只能写简单表达式，不能写多行。
