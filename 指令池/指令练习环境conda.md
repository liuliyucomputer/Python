

conda create --prefix G:\Python\env_conda_python12 创建conda环境

`prefix` 表示“前缀目录”


`--prefix`（或 `-p`）参数用于指定 Conda 环境在磁盘上的“安装前缀路径”，也就是创建环境时所有解释器、库、脚本会放置的绝对目录。

* 中文：`prefix` 表示“前缀目录”，常用于把环境安装到自定义路径，例如 `G:\Python\env_conda_python12`。
* English：`prefix` means the “installation prefix directory,” i.e., the exact folder path where Conda drops the interpreter, packages, and scripts.

“环境名 = G:\Python\env_conda_python12”

激活：**conda**activate**G:**\P**ython**\e**nv_conda_python12**
