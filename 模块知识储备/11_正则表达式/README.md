# 正则表达式

正则表达式（Regular Expression）是一种用于匹配字符串中字符组合的模式，是一种强大的文本处理工具。在Python中，正则表达式主要通过`re`模块实现。

## 模块概述

### re模块

`re`模块是Python标准库中用于处理正则表达式的模块，提供了以下功能：

- **正则表达式编译**：将正则表达式编译为模式对象，提高匹配效率
- **字符串匹配**：检查字符串是否匹配正则表达式
- **字符串搜索**：在字符串中搜索匹配正则表达式的内容
- **字符串替换**：替换字符串中匹配正则表达式的内容
- **字符串分割**：根据正则表达式分割字符串
- **分组匹配**：提取正则表达式匹配的分组内容

### 正则表达式基础

正则表达式的基本语法包括：

- **字符类**：`[abc]`匹配a、b或c；`[^abc]`匹配除a、b、c之外的字符
- **预定义字符类**：`\d`匹配数字；`\w`匹配字母、数字或下划线；`\s`匹配空白字符
- **数量词**：`*`匹配0次或多次；`+`匹配1次或多次；`?`匹配0次或1次；`{n}`匹配n次；`{n,}`匹配n次或多次；`{n,m}`匹配n到m次
- **边界匹配**：`^`匹配字符串开头；`$`匹配字符串结尾；`\b`匹配单词边界
- **分组**：`(...)`匹配括号内的表达式，并捕获匹配的内容
- **选择**：`|`匹配|左边或右边的表达式
- **转义**：`\`转义特殊字符

### 正则表达式高级用法

正则表达式的高级功能包括：

- **命名分组**：`(?P<name>...)`给分组命名，便于后续引用
- **非捕获分组**：`(?:...)`匹配括号内的表达式，但不捕获匹配的内容
- **正向预查**：`(?=...)`匹配后面跟着...的内容
- **负向预查**：`(?!...)`匹配后面不跟着...的内容
- **回溯引用**：`\1`、`\2`等引用之前捕获的分组内容
- **贪婪匹配与非贪婪匹配**：`*?`、`+?`、`??`等非贪婪匹配模式

### 实际应用示例

正则表达式在实际项目中的应用包括：

- **数据验证**：验证邮箱、电话号码、身份证号等格式
- **数据提取**：从文本中提取URL、IP地址、日期等信息
- **数据清洗**：去除文本中的特殊字符、空格等
- **文本替换**：批量替换文本中的特定内容
- **日志分析**：从日志文件中提取有用信息
- **网页爬虫**：从HTML文本中提取所需内容

## 注意事项

1. **正则表达式的性能**：复杂的正则表达式可能会影响性能，应尽量保持正则表达式的简洁
2. **正则表达式的可读性**：复杂的正则表达式可读性较差，应添加适当的注释
3. **正则表达式的转义**：在Python字符串中使用正则表达式时，需要注意转义字符的处理
4. **正则表达式的测试**：使用在线正则表达式测试工具或Python的re模块测试正则表达式
5. **正则表达式的兼容性**：不同编程语言的正则表达式语法可能略有不同，注意兼容性问题
